flex 基本概念

父容器

使用 flex 布局首先要设置父容器 display: flex，

主轴：justify-content:

flex-start（默认值） flex-end center space-between space-around

交叉轴：align-items: 需要子元素有宽高

flex-start（默认值） flex-end center baseline stretch

设置换行方式：flex-wrap:

wrap nowrap（默认值） wrap-reverse

多行沿交叉轴对齐：align-content: 当子容器多行排列时，设置行与行之间的对齐方式。

flex-start flex-end center space-around space-between stretch（默认值）

子容器

使用 flex 布局首先要设置父容器 display: flex

主轴：flex: 有flex不会换行,优先级高于width

flex-grow flex-shrink flex-basis

交叉轴：align-self: 能覆盖父容器的align-items

flex-start flex-end center baseline stretch(默认值)
